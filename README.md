# 考研408每日真题推送插件

为 [AstrBot](https://github.com/AstrBot/AstrBot) 开发的考研408历年真题推送插件。

## 功能介绍

- **随机抽奖模式**：`/408抽奖` 随机从题库抽一道真题，像开盲盒一样刺激学习兴趣
- **编号调取模式**：`/408-122` 直接调取第122道题目，精准定位
- **答案附带**：所有抽题命令默认直接输出答案和详细解析，不需要额外输入查答案
- **定时推送**：支持群订阅，每天定时自动推送一道真题

## 安装方法

1. 将 `astrbot_plugin_daily408` 文件夹复制到 AstrBot 的插件目录（如 `data/plugins/`）
2. 重启 AstrBot 或重载插件
3. 发送 `/408菜单` 即可开始使用

## 命令列表

| 命令 | 说明 |
|------|------|
| `/408菜单` | 显示主菜单（私聊/群聊均可） |
| `/408帮助` | 显示详细帮助（私聊/群聊均可） |
| `/408抽奖 [科目]` | 随机抽一题，科目可选：ds/co/os/cn |
| `/408-<编号>` | 按编号调取指定题目，如 `/408-3`、`/408-122` |
| `/408订阅` | 当前群订阅每日推送（需群聊+管理员） |
| `/408退订` | 当前群取消订阅（需群聊+管理员） |
| `/408列表` | 查看当前群订阅状态（需群聊+管理员） |
| `/408全部订阅` | 管理员查看所有订阅群 |

## 科目代码

- `ds` — 数据结构
- `co` — 计算机组成原理
- `os` — 操作系统
- `cn` — 计算机网络

命令中既可以用代码，也可以用中文名称，例如：
- `/408抽奖 ds`
- `/408抽奖 数据结构`
- `/408-5`

## 扩充题库

题库文件位于插件目录下的 `data/questions.json`，按照以下格式添加新题即可：

```json
{
  "id": "ds_2010_1",
  "year": 2010,
  "subject": "数据结构",
  "subject_code": "ds",
  "type": "单选题",
  "question": "题目内容",
  "options": [
    "A. 选项1",
    "B. 选项2",
    "C. 选项3",
    "D. 选项4"
  ],
  "answer": "C",
  "analysis": "解析内容"
}
```

### 字段说明

| 字段 | 说明 |
|------|------|
| `id` | 唯一标识，建议格式：`subject_code_年份_序号` |
| `year` | 真题年份 |
| `subject` | 科目中文名 |
| `subject_code` | 科目代码：ds / co / os / cn |
| `type` | 题型，如：单选题、综合应用题 |
| `question` | 题目内容 |
| `options` | 选项列表（没有可留空数组） |
| `answer` | 正确答案 |
| `analysis` | 详细解析 |

**编号规则**：插件会自动按 `questions.json` 数组中的**顺序**给每道题编号（从1开始）。你只需按顺序追加新题，`/408-<编号>` 就能自动对应。

## 配置项

在 AstrBot 的插件配置中可修改：

- **推送时间**：默认每天 `09:00`
- **管理员列表**：额外指定插件管理员
